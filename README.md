# Fackge
## 是 Fackge?
这是一款基于C++开发的脚本语言,十分简单,没有语法树,没有栈,全靠if嵌套,是不是很牛.
## 怎么使用fackge?
下载我们的编译文件,双击运行.
### 对Windows用户
下载我们的exe可执行文件使用
### 对Linux用户
下载我们的源码进行编译
